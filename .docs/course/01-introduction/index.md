# Module 1: Introduction

Set up your development environment and understand the foundations of nanopayments.

## Lessons

1. **Getting Started** — Course overview, what you'll build, and how the modules connect
2. **What Are Nanopayments?** — The deposit → pay → settle flow and real-world use cases
3. **Understanding USDC** — Decimal precision, Arc Testnet details, and working with amounts in code
4. **Setting Up Your Environment** — Project initialization, dependencies, folder structure, and `.env` setup
5. **Testing Your Setup** — Verify all dependencies are installed and the project runs
6. **Troubleshooting** — Common setup issues and how to fix them

## Key Concepts

- Nanopayments enable sub-cent, pay-per-use pricing via batched settlement
- USDC has 6 decimal places — the smallest unit is $0.000001
- Arc Testnet uses USDC as both the payment token and gas token
